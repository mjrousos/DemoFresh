# Copilot CLI Prompts

These are the prompts and actions used to generate DemoFresh using the Copilot CLI and SDK.

1. Created initial requirements.md
1. `I've captured requirements for the DemoFresh app in @requirements.md . Please take a look and let me know what other details I should add there to enable you to create a high-quality implementation plan.`
1. Answer questions about requirements and add details as needed.
1. Review the generated plan and make any adjustments needed to ensure it meets the requirements and is feasible to implement.
1. `Please double check that the generated plan completely addresses all the requirements outlined in @requirements.md and that there are no gaps. Let me know if you identify any gaps or areas that need further clarification.`
1. `Please copy the generated plan to this repo and save it as plan.md .`
1. `Accept plan and build on autopilot using fleet mode.`
